About

WheelieNames is a browser-based random name picker wheel built on the Web Crypto API for cryptographically secure, mathematically fair selection. Every spin uses crypto.getRandomValues() — the same standard behind HTTPS and banking — and all data stays in your browser via localStorage. No accounts, no tracking, no data collection.
